# RuT in English
RuTracker translated into English.

## Installation

1. Make sure you have script enabled in your browser (these instructions refer to the latest versions of the browser):

	* [Tampermonkey](https://www.tampermonkey.net/) (proprietary)
	* [Violentmonkey](https://violentmonkey.github.io/get-it/)
	* [Greasemonkey](https://addons.mozilla.org/firefox/addon/greasemonkey/) (some scripts are **not compatible with Greasemonkey 4**!)

## Updating

This script is set up to automatically update. You can check for updates from within the Greasemonkey or Tampermonkey menu, or click on the install link again to get the update.
